# Stock Prediction System

This is a stock prediction system built with Python and Streamlit. It fetches historical stock data, trains a simple model, and predicts the stock's next day price.

## Features
- Displays historical stock data
- Adds technical indicators for feature engineering
- Trains a Linear Regression model
- Predicts next-day prices for a given stock ticker
